# Houraiji 个人博客内容包

这套内容基于公开资料、简历信息和 GitHub 公开仓库整理，采用公开安全口径：不展示真实姓名、敏感联系方式和私人信息；保留适合博客展示的学校专业、学习阶段、项目和联系方式。

## 文件说明

| 文件 | 用途 |
| --- | --- |
| `profile.md` | 个人资料表单，可用于 About 页面 |
| `growth-route.json` | 2023 到 2026 的成长路线 JSON |
| `projects.json` | 项目档案 JSON，覆盖公开仓库、实习项目和补充叙事 |
| `articles/` | 27 篇 Markdown 文章，包含技术复盘、日常文章和短篇小说 |

## 建议导入顺序

1. 先导入 `profile.md`，作为个人主页资料。
2. 再导入 `growth-route.json`，用于时间线组件或成长路线页面。
3. 导入 `projects.json`，用于项目列表、项目卡片或作品集页面。
4. 最后导入 `articles/`，按日期发布文章。
